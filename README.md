# Deaf Student Solutions Website

Website redesign for Deaf Student Solutions Ltd — three design options to choose from.

Open the HTML files in a browser to compare designs:

- **Style A (Modern Minimal)** — Clean white, blue accent, Inter + Plus Jakarta Sans
- **Style B (Warm Professional)** — Cream/burgundy/gold, Playfair Display + Source Sans 3
- **Style C (Bold Contemporary)** — Dark mode, emerald/teal, Space Grotesk + DM Sans

All files are self-contained (inline CSS + JS, Google Fonts via CDN). No build step needed.
